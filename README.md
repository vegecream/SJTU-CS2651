### 简介
这是我在学习CS2651课程中完成的Lab3作业。由于糟糕的描述与instrction，考虑到作业本身没有提供任何的测试用例，很多人会参考一些其他的答案。传承上的实现很不错且总体正确，但是在lab3的一些实现中有误，例如一些命令的代号有问题，以及，实现思路使用了繁复的十进制与二进制转换。基于这些原因，我重写了这些代码并**手动编写**了所有的测试用例。
